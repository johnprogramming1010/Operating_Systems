Description of programs
I have 6 files included in this folder.
I have 1 server class and 5 client classes.

Steps to run program: 
1. put all of the files in one folder to run on command prompt. 
2. use cd/ to clear your directory and then use cd and drag the folder from desktop into the command prompt.
3. In the first command Prompt run James_server.java and execute the program
4. Open up 2 other commands prompts and run James_client and James_client2 to execute.
5. After they are both running, add other clients up to 5 as you wish.
6. to leave the chat type in, "@exit".
